#  Sports Day Scoreboard

## Environment
| Package | Version |
| --- | --- |
| ruby | 3.2.2 |
| Postgres | Any |

## Development Quickstart
1. Copy `.env.development.example` to `.env`
2. Once you start the docker container (`docker compose up`)
3. Setup the database (`docker compose exec web rails db:setup`)
4. Success!
